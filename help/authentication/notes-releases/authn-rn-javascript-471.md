## Panoramica sulla versione {#release-overview-471}

* Rimossa la versione 3.5.0 di AccessEnabler JavaScript SDK.

L’URL di staging è: https://entitlement.auth-staging.adobe.com/entitlement/v4/AccessEnabler.js
